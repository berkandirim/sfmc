# SFMC [![Build Status](https://travis-ci.org/berkandirim/sfmc.svg?branch=master)](https://travis-ci.org/berkandirim/sfmc)

## To run this project locally

Make sure mongoDB is running
```
$ mongod
```

```
$ npm install
```

If you have nodemon
```
$ nodemon
```

Else
```
$ npm run start
```

Then go to http://localhost:3000
